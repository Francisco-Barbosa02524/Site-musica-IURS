# Projeto de Música

## Descrição

Este é um projeto de música que permite criar e editar músicas utilizando instrumentos virtuais e áudios.

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Node.js
* Express.js
* Tone.js
* Web Audio API

## Instalação e configuração

1. Clone o repositório com `git clone https://github.com/seu-nome/projeto-musica.git`
2. Instale as dependências com `npm install`
3. Execute o projeto com `node app.js`

## Uso e funcionalidades

* Crie músicas utilizando instrumentos virtuais e áudios
* Edite músicas existentes
* Grave áudios com microfone
